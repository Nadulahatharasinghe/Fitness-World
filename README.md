# 🏋️ Fitness World Web System 💪

## 📌 Project Overview

**Fitness World** is a web-based application developed as part of the Information Technology Project (ITP). The system is designed to help users manage their fitness journey by tracking workouts, exploring fitness programs, and maintaining healthy lifestyle habits through a modern digital platform.

This project demonstrates full-stack development using the **MERN stack**, following software engineering best practices suitable for academic evaluation and real-world applications.

---

## 🎯 Objectives

* Provide a digital platform for fitness tracking and management
* Help users explore workout plans and fitness programs
* Enable user authentication and personalized experiences
* Apply full-stack development concepts using modern technologies
* Practice team collaboration and Git version control

---

## 🧩 Architecture

* **MERN Stack Architecture**

  * React → Frontend UI
  * Node.js + Express → Backend API
  * MongoDB → Database
  * RESTful APIs → Communication between client and server

---

## ✨ Key Features

* User registration and login (authentication & authorization)
* User profile management
* Browse workout plans and fitness programs
* Track workouts and fitness progress
* Admin panel:

  * Add workout plans
  * Update workout details
  * Delete records
* Responsive and user-friendly UI
* Secure API handling with validation

## 🚀 How to Run the Project

### Prerequisites

* Node.js installed
* MongoDB installed and running
* Git installed

### Steps

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project folder
cd Fitness-World

# Install backend dependencies
cd server
npm install

# Start backend server
npm start
```

In a new terminal:

```bash
# Navigate to frontend
cd client

# Install frontend dependencies
npm install

# Start React app
npm start
```
